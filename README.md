# go-pexels

The unofficial Pexels API client written in Go.

## Supported Endpoints & Features

### Photo Endpoints

- [x] Search Photos
- [x] Get Curated Photos
- [x] Get Photo by ID

### Video Endpoints

- [x] Search Videos
- [x] Get Popular Videos
- [x] Search Videos

### Collection Endpoints

- [x] Get User's Collection
- [x] Get User's Collections
